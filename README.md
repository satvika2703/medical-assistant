# Mindful Dose — AI Medication Reminder (Streamlit)

A calm, daily check-in web app. Each day, you tell it whether you took your
medication. If you've missed a few in a row, an AI agent (Gemini) writes a
gentle nudge — and if it gets serious, it notifies a family member.

## Features

- Clean, custom-styled UI (warm "wellness" theme — not a default Streamlit look)
- Daily check-in: "I took it" / "Not yet / missed"
- 14-day streak visualization
- AI-generated reminder messages via Gemini (falls back to rule-based if no key)
- Optional real email alerts to a family member after 4+ missed doses
- Multi-user — switch between profiles in the sidebar
- SQLite storage (medication.db), created automatically

## Run locally

```bash
pip install -r requirements.txt
streamlit run streamlit_app.py
```

Open the URL it prints (usually http://localhost:8501).

### Enable AI messages (optional but recommended)

1. Get a free key at https://aistudio.google.com/app/apikey
2. Copy `.env.example` to `.env`
3. Set `GEMINI_API_KEY=your_key`

Without a key, the app still works fully — it just uses pre-written
rule-based messages instead of AI-generated ones.

### Enable family email alerts (optional)

1. Create a Gmail App Password: https://myaccount.google.com/apppasswords
   (requires 2-Step Verification enabled)
2. In `.env`, set `SENDER_EMAIL` and `SENDER_PASSWORD`
3. When creating a profile in the app, fill in the "Family member's email" field

Without this configured, family notifications are simulated (shown in the
activity log) instead of sent.

## Deploy on Streamlit Community Cloud (free)

1. Push this folder to a public GitHub repository.
2. Go to https://share.streamlit.io and sign in with GitHub.
3. Click **New app**, pick your repo and branch, and set the main file path
   to `streamlit_app.py`.
4. Before (or after) deploying, add your secrets:
   - In the app dashboard, go to **Settings → Secrets**
   - Paste in (with your real values):
     ```toml
     GEMINI_API_KEY = "your_gemini_key_here"
     SENDER_EMAIL = "your_gmail@gmail.com"
     SENDER_PASSWORD = "your_app_password_here"
     ```
5. Click **Deploy**. Your app gets a public URL like
   `https://your-app-name.streamlit.app`.

### Important note on storage

Streamlit Community Cloud's filesystem is **ephemeral** — the SQLite database
(`medication.db`) will reset whenever the app restarts or redeploys (e.g.
after inactivity, or a new git push). This is fine for demos and personal
testing. For persistent multi-day tracking in production, swap `memory.py`
to use a hosted database (e.g. Supabase, Turso, or any Postgres/SQLite-over-HTTP
service) — the function signatures in `memory.py` are designed so this is a
drop-in change; nothing else in the app needs to know.

## How the agent decides

| Missed doses in a row | Action          | What happens                          |
|------------------------|-----------------|----------------------------------------|
| 0–1                     | NORMAL_REMINDER | Gentle on-screen reminder               |
| 2–3                     | URGENT_ALERT    | More direct on-screen message           |
| 4+                      | NOTIFY_FAMILY   | On-screen alert + email to family member|

Messages are generated by Gemini when an API key is configured (varied,
personalized wording each time); otherwise a fixed rule-based message for
each tier is used — the *thresholds* are the same either way.
